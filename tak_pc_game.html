Description

Tak is a two-player abstract strategy game designed by James Ernest and Patrick Rothfuss.
I'm currently working on this PC rendition of the game using Unity.
Current features:
    - Create a game, host it on your computer (use your PC as a host server).
    - Join another player's game using their public IP address.
    - Select from 3x3, 4x4, 5x5, 6x6, and 8x8 game board modes.
    - Movement by clicking and dragging, zooming with scroll-wheel.
    - Place stones, either as regular or standing stones.

    - Capstones included and functional in board sizes above 3x3.
    - Move stones around the board, onto other stones, and move stacks of stones together.
    - Limitations for stone placement and movements based on original game rules.
Features currently in development:

    - Audio effects, and corresponding sound settings menu.
    - Visual cues and effects for UX.
    - Find and fix player piece movement bugs.

The only thing left before this game could be considered functionally complete is testing, and any modifications to algorithms that arise from said testing. Movement bugs appear to exist.

The game can be downloaded (below) and played without issue, but the online multiplayer is extremely clunky and difficult to use. However, barring a likely few bugs, the game is playable.
